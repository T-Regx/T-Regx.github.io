T-Regx also allows to split a string by regular expression.

<CodeTabs
    tregx="pattern('[ ,]{2}')->split('Joffrey, Cersei, Ilyn Payne, The Hound');"
    php="preg::split('/[ ,]{2}/', 'Joffrey, Cersei, Ilyn Payne, The Hound');"/>
<Result>['Joffrey', 'Cersei', 'Ilyn Payne', 'The Hound']</Result>

## Including the delimiter

To include a delimiter in the results, enclose it with a capturing group.

<CodeTabs
    tregx="pattern('([ ,]{2})')->split('Joffrey, Cersei, Ilyn Payne');"
    php="preg::split('/([ ,]{2})/', 'Joffrey, Cersei, Ilyn Payne', -1, PREG_SPLIT_DELIM_CAPTURE);"/>
<Result>['Joffrey', ', ', 'Cersei', ', ', 'Ilyn Payne']</Result>
