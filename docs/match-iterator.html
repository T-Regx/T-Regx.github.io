Iterator

T-Regx uses vanilla PHP Iterator with standard methods:

  • current()
  • key(): scalar
  • next(): void
  • rewind(): void
  • valid(): bool

Iterator to array

Method match()->iterator() returns an implementation of PHP Iterator which you can be used with other Iterator methods or your own code, for example PHP methods iterator_to_array() and iterator_count().

T-Regx
PHP
$iterator = pattern('\w+')->match('Apples are cool')->iterator();

return iterator_to_array($iterator);
preg::match_all('/\w+/', 'Apples are cool', $matches);
$iterator = new ArrayIterator($matches[0]);

return iterator_to_array($iterator);
['Apples', 'are', 'cool']
